Περιγραφή

Στο πρόγραμμα θα συμπληρώσετε την μέθοδο compute στην κλάση Factorial στο πακέτο gr.duth.ee.euclid.datastructures.factorial.

Η μέθοδος πρέπει να επιστρέφει ένα long με τιμή το παραγοντικό της εισόδου x. Ορισμένα παραδείγματα φαίνονται στον παρακάτω πίνακα.

Είσοδος x12510
Έξοδος121203628800

Σημείωση: η είσοδος θα είναι μεγαλύτερη από μηδέν και θα είναι τέτοια ώστε να μην προκαλεί υπερχείλιση.
