Αναζήτηση διαδοχικών ακεραίων σε πίνακα ακεραίων.

Σύνοψη

Σε αυτή την άσκηση πρέπει να γράψετε κώδικα που θα πραγματοποιεί αναζήτηση για διαδοχικούς φθίνοντες ακεραίους σε έναν πίνακα ακεραίων.

Περιγραφή

Στο πρόγραμμα θα συμπληρώσετε την μέθοδο contiguousIntegerSearch στην κλάση ContiguousIntegerSearch στο πακέτο gr.duth.ee.euclid.datastructures.contiguous_integer_search.

Η μέθοδος πρέπει να επιστρέφει έναν ακέραιο (int) που αποτελεί τη θέση έναρξης (η θέση του πρώτου στοιχείου) της ομάδας διαδοχικών ακεραίων τιμών που αναζητείται μέσα στον πίνακα (η θέση αρχίζει από το μηδέν).

Αν δεν βρεθεί μια ομάδα διαδοχικών φθινόντων ακεραίων κατάλληλου μήκους πρέπει να επιστραφεί η θέση -3.

Μια διαδοχική ομάδα n φθινόντων ακεραίων αποτελείται από ένα τμήμα του πίνακα με τιμές k, k-1, k-2, ..., k - (n-1) σε διαδοχικές θέσεις του πίνακα i, i+1, i+2, ..., i + n-1.

  • Ο πίνακας που δίνεται ως όρισμα θα έχει πάντα μέγεθος μεγαλύτερο ή ίσο του 5 και δεν χρειάζεται να ελέγξετε περιπτώσεις όπου το μέγεθος είναι 5 ή το όρισμα null.
  • Ο πίνακας που δίνεται ως όρισμα μπορεί να έχει πολλαπλές φορές το ίδιο στοιχείο και θα πρέπει ο αλγόριθμος να δουλεύει και σε αυτές τις περιπτώσεις.
  • Το απαιτούμενο μέγεθος ομάδας ακεραίων δίνεται ως παράμετρος και είναι μεγαλύτερη ή ίση του 2 και πάντα μικρότερη ή ίση του μεγέθους του πίνακα.

Είσοδος

  1. Ένας μη κενός πίνακας ακεραίων (int[]) με τιμές στοιχείων στο κλειστό διάστημα [-9999, 9999] και μεγέθους Ν, με 5 ≤ Ν ≤ 1000.
  2. Ένας ακέραιος (int) στο κλειστό διάστημα [2, N] ο οποίος προσδιορίζει το απαιτούμενο μέγεθος της της ομάδας διαδοχικών ακεραίων που πρέπει να βρεθεί.

Έξοδος

Η θέση του του πρώτου στοιχείου της πρώτης ευρεθείσας ομάδας φθινόντων ακεραίων (αρχίζοντας από την αρχή του πίνακα) ζητούμενου μεγέθους ή -1 αν δεν βρίσκεται μια τέτοια ομάδα εντός του πίνακα.

Επισημάνσεις / Διευκρινήσεις

Θα πρέπει ο αλγόριθμός σας να εντοπίζει ομάδα με K ή περισσότερα στοιχεία. Αν, για παράδειγμα, είσοδος είναι ο πίνακας:

Θέση 0 1 2 3 4 5 6 7
Τιμή 7 12 9 16 15 14 13 20

Τότε για τα ζητηθέντα μεγέθη ομάδας, οι απαντήσεις είναι:

Ζητηθέν Μέγεθος Ομάδας Σωστή Απάντηση / Θέση Πρώτου Στοιχείου Ομάδας
2 3
3 3
4 3
5 -1

Παράδειγμα

int[] values = new int[]{3, 4, 4, 3, 2, 12};//values[0]=3, values[1]=4, values[2]=4, values[3]=3, values[4]=2, values[5]=12

int valueLengthToFind1 = 3;
int valuePosition1 = gr.duth.ee.euclid.datastructures.contiguous_integer_search.ContiguousIntegerSearch.contiguousIntegerSearch(values, valueLengthToFind1);
assert(valuePosition1 == 2); // 4, 3, 2

int valueLengthToFind2 = 4;
int valuePosition2 = gr.duth.ee.euclid.datastructures.contiguous_integer_search.ContiguousIntegerSearch.contiguousIntegerSearch(values, valueLengthToFind2);
assert(valuePosition2 == -1); // Not found
