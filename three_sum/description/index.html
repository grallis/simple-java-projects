Το πρόβλημα 3SUM.

Σύνοψη

Σε αυτή την άσκηση πρέπει να γράψετε κώδικα που θα λύνει το πρόβλημα 3SUM.

Περιγραφή

Στο πρόγραμμα θα συμπληρώσετε την μέθοδο compute στην κλάση ThreeSum στο πακέτο gr.duth.ee.euclid.datastructures.three_sum.

Η μέθοδος πρέπει να επιστρέφει ένα int με τιμή το πλήθος των διακριτών τριάδων αριθμών που ανήκουν στον πίνακα εισόδου και των οποίων το άθροισμα είναι μηδέν. Ορισμένα παραδείγματα παρουσιάζονται παρακάτω.

Είσοδος: {-1,0,2,1,-3}

Οι διαφορετικές τριάδες που αθροίζουν σε 0 είναι οι: {-3,1,2} και {-1,0,1} οπότε η σωστή απάντηση είναι 2.

Είσοδος: {1,-4,2,-3,2}

Οι διαφορετικές τριάδες που αθροίζουν σε 0 είναι οι: {-4,2,2} και {-3,1,2} οπότε η σωστή απάντηση είναι 2.

Είσοδος: {-3,1,-1,3,2,0}

Οι διαφορετικές τριάδες που αθροίζουν σε 0 είναι οι: {-3,0,3}, {-3,1,2} και {-1,0,1} οπότε η σωστή απάντηση είναι 3.

Σημείωση: η είσοδος θα είναι ένας πίνακας περίπου 25 στοιχείων στο κλειστό διάστημα [-10,10].
